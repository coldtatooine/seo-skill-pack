# SEO Pack — Claude Plugin Marketplace

> 7 SEO skills for local audits, AI search readiness, GEO content briefs, meta optimization, schema markup, competitor gap analysis, and site health audits. Built for Claude Cowork and Claude Code.

## Skills

| Skill | What it does |
|---|---|
| `local-seo-audit` | Full local SEO audit — GBP, NAP consistency, citations, on-page signals |
| `ai-search-readiness` | Audit site readiness for AI-powered search engines (ChatGPT, Perplexity, Gemini) |
| `geo-content-brief` | Content briefs optimized for traditional SEO and Generative Engine Optimization |
| `meta-optimizer` | Generate optimized title tags and meta descriptions with CTR-focused formulas |
| `local-schema-markup` | Generate and validate LocalBusiness, FAQ, HowTo, and other schema types |
| `competitor-gap-analyzer` | Identify keyword and content gaps vs top competitors |
| `site-health-audit` | Technical SEO health check — Core Web Vitals, crawlability, indexability |

## Installation

### Claude Cowork (recommended)

1. Open **Customize** (bottom-left)
2. Go to **Browse plugins** → **Personal** → **+**
3. Select **Add marketplace from GitHub**
4. Enter: `coldtatooine/seo-skill-pack`

All skills install automatically.

### Claude Code (CLI)

```bash
# Add the marketplace
claude plugin marketplace add your-github-username/seo-skill-pack

# Install the plugin
claude plugin install seo-skill-pack@seo-skill-pack
```

- `Run a local SEO audit for my plumbing business in São Paulo`
- `Check NAP consistency for [business name] across the web`

**AI Search Readiness:**
- `Is my site ready to be cited by ChatGPT and Perplexity?`
- `Audit my homepage for AI search visibility`

- `Create a content brief for "best tile stores in [city]"`

**Meta Tags:**
- `Optimize the title and meta description for my product pages`
- `Generate meta tags for these 10 category pages [attach CSV]`

**Schema Markup:**
- `Generate LocalBusiness schema for my store`
- `Create FAQ schema from these 8 questions`

**Competitor Gap Analysis:**
- `What keywords are my competitors ranking for that I'm not?`
- `Identify content gaps between my site and [competitor URL]`

**Site Health:**
- `Run a technical SEO audit on my site`
- `What are the biggest crawlability issues I should fix first?`
