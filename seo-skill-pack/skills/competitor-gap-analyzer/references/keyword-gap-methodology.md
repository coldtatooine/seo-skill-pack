# Keyword Gap Analysis Methodology

## Overview

Keyword gap analysis identifies search queries where competitors rank but the target business does not, revealing untapped opportunities for organic traffic growth. This methodology works with both paid SEO tools and free manual techniques.

---

## Step 1: Define the Keyword Universe

### Identify Seed Keywords
Start with 5-10 core topics that define the business:
1. Primary service/product keywords
2. Industry category keywords
3. Location-modified keywords (for local businesses)
4. Problem/solution keywords customers search for
5. Brand and competitor brand keywords

### Expand with Modifiers
For each seed keyword, create variations:
- **Intent modifiers**: best, top, how to, what is, cost, price, near me, reviews
- **Location modifiers**: [city], [state], [neighborhood], near [landmark]
- **Qualifier modifiers**: affordable, professional, emergency, free, fast, certified
- **Comparison modifiers**: vs, alternative, comparison, review, pros and cons
- **Question modifiers**: how, what, why, when, where, which, can, does, is

---

## Step 2: Map Competitor Keywords

### Manual SERP Analysis
For each keyword in the universe:
1. Search the keyword in Google (use incognito/private browsing)
2. Record positions 1-10 for each competitor
3. Note SERP features present (featured snippet, PAA, local pack, AI overview)
4. Record which competitor holds each featured position

### Using Google Search Operators
```
site:competitor.com "keyword"          → Check if competitor has content
site:competitor.com intitle:"keyword"  → Check if keyword is in titles
site:competitor.com inurl:keyword      → Check URL structure
```

### Using Google Search Console (for own site)
- Performance > Search Results > Queries
- Filter by position (page 2+ = weak keywords)
- Sort by impressions (high impressions + low CTR = opportunity)

---

## Step 3: Classify Keyword Gaps

### Gap Categories

#### 1. Missing Keywords (Highest Priority)
- **Definition**: Competitor ranks in top 10, business has no ranking
- **Signal**: Complete content gap
- **Action**: Create new content targeting this keyword
- **Priority**: By estimated volume × relevance

#### 2. Weak Keywords (Critical)
- **Definition**: Business ranks page 2-3 (positions 11-30), competitor ranks page 1
- **Signal**: Content exists but needs improvement
- **Action**: Optimize existing content or create better version
- **Priority**: By current position (closer to page 1 = easier win)

#### 3. Declining Keywords (High)
- **Definition**: Business previously ranked higher, competitors now outrank
- **Signal**: Content decay or competitor improvement
- **Action**: Update, expand, or rewrite content
- **Priority**: By traffic lost

#### 4. Untapped Long-Tail (Medium)
- **Definition**: Low-competition, specific queries competitors target
- **Signal**: Niche content opportunity
- **Action**: Create targeted content or add sections to existing pages
- **Priority**: By relevance × ease of ranking

#### 5. Question Keywords (Medium)
- **Definition**: Questions appearing in PAA/FAQs that competitors answer
- **Signal**: Featured snippet opportunity
- **Action**: Add FAQ sections or dedicated answer pages
- **Priority**: By volume × featured snippet potential

---

## Step 4: Prioritize Opportunities

### Scoring Framework

Score each keyword opportunity (1-10 for each factor):

| Factor | Weight | Description |
|--------|--------|-------------|
| **Relevance** | 30% | How closely does the keyword align with business offerings? |
| **Volume** | 25% | Estimated search volume (use Google Keyword Planner ranges) |
| **Competition** | 20% | How hard to rank? (fewer competitors in top 10 = easier) |
| **Intent Match** | 15% | Does the searcher intent match what the business offers? |
| **Current Position** | 10% | Existing ranking (closer = easier to improve) |

**Priority Score = (Relevance × 0.3) + (Volume × 0.25) + (Competition × 0.2) + (Intent × 0.15) + (Position × 0.1)**

### Priority Matrix

| Score Range | Priority | Timeline |
|-------------|----------|----------|
| 8-10 | Critical — Do First | Week 1-2 |
| 6-7.9 | High — Do Soon | Weeks 3-4 |
| 4-5.9 | Medium — Plan for | Months 2-3 |
| 2-3.9 | Low — Backlog | When resources allow |
| 0-1.9 | Skip | Not worth pursuing |

## Step 5: Action Planning

### For Missing Keywords
1. Research what format/content currently ranks
2. Create a content brief (use GEO Content Brief skill)
3. Produce content that matches or exceeds competitor quality
4. Add internal links from existing relevant pages
5. Submit to Google Search Console for indexing
6. Monitor rankings over 4-8 weeks

### For Weak Keywords
1. Analyze why competitor content ranks higher
2. Identify content gaps (word count, depth, format, freshness)
3. Update existing content:
   - Expand sections that are thin
   - Add missing subtopics
   - Update statistics and dates
   - Improve heading structure
   - Add visual content (tables, charts)
4. Improve on-page SEO (title, meta, headings)
5. Add internal links pointing to the page
6. Monitor position changes

### For Declining Keywords
1. Check Google Search Console for position drops
2. Compare content against new top-ranking pages
3. Look for technical issues (speed, mobile, indexing)
4. Refresh content with new information and data
5. Re-earn or build new links to the page

### For Question Keywords
1. Add FAQ schema to existing relevant pages
2. Create dedicated answer sections
3. Use the exact question as an H2 or H3
4. Provide a concise 2-3 sentence answer immediately after the heading
5. Follow with expanded detail

## Competitive Intelligence Sources (Free)

| Source | What It Reveals |
|--------|----------------|
| Google Search | Current rankings, SERP features |
| Google Search Console | Own keyword data, CTR, positions |
| Google Keyword Planner | Volume ranges, competition levels |
| Google Trends | Trend direction, seasonal patterns |
| Google Alerts | New competitor content/mentions |
| Social media | Content strategy, engagement levels |
| Competitor sitemaps | Full content inventory |
| Competitor blog/RSS | Publishing frequency, topics |
| Press releases | Business developments, campaigns |
| Job postings | Strategic direction (hiring content writers = content push) |
