# Meta Title Tag Formulas by Page Type & Industry

## Universal Title Formulas

### Homepage
```
[Primary Keyword] — [Value Proposition] | [Brand Name]
[Brand Name] — [Primary Service] in [Location]
[Primary Keyword] [Modifier] — [Brand Name]
```

### Blog Post / Article
```
[Number] [Keyword] [Promise] ([Year])
How to [Keyword]: [Benefit] ([Year])
[Keyword]: [Definitive Statement] | [Brand]
What Is [Keyword]? [Concise Answer] | [Brand]
[Keyword] vs [Keyword]: [Verdict] ([Year])
The [Adjective] Guide to [Keyword] ([Year])
```

### Product Page
```
[Product Name] — [Key Feature] | [Brand]
Buy [Product] — [Benefit] | [Brand] [Offer]
[Product Name]: [Specification] | [Brand]
```

### Category / Collection Page
```
[Category Keyword] — [Count] [Items] | [Brand]
Shop [Category] — [Differentiator] | [Brand]
[Category Keyword] for [Audience] | [Brand]
```

### Service Page
```
[Service Keyword] in [City] — [Differentiator] | [Brand]
Professional [Service] — [Benefit] | [Brand]
[Service] Services — [Trust Signal] | [Brand]
```

### Location / Local Page
```
[Service] in [City, State] — [Differentiator] | [Brand]
[City] [Service Provider] — [Trust Signal] | [Brand]
Best [Service] in [City] — [Qualification] | [Brand]
```

### Comparison Page
```
[A] vs [B]: [Winner/Verdict] ([Year]) | [Brand]
[A] vs [B] — Which Is [Adjective]? | [Brand]
[A] vs [B]: Honest Comparison [Year] | [Brand]
```

### FAQ Page
```
[Topic] FAQ — [Count] Common Questions Answered | [Brand]
[Topic]: Your Questions Answered | [Brand]
```

## CTR-Boosting Modifiers

### Numbers & Lists
- "7 Ways to...", "10 Best...", "5 Steps to..."
- Numbers in titles increase CTR by 15-20% on average

### Year
- "(2026)", "[Updated 2026]", "in 2026"
- Signals freshness; especially important for how-to and "best of" content

### Power Words (by category)

**Urgency**: Now, Today, Quick, Fast, Instant, Immediate
**Value**: Free, Save, Affordable, Budget, Cheap, Discount
**Authority**: Expert, Professional, Proven, Certified, Official, Trusted
**Exclusivity**: Secret, Hidden, Insider, Exclusive, Limited
**Emotion**: Amazing, Incredible, Essential, Critical, Ultimate
**Comparison**: Best, Top, vs, Compared, Review, Honest
**How-to**: Guide, Tutorial, Step-by-Step, Complete, Easy
**Question**: How, What, Why, When, Where, Which

### Brackets & Parentheses
- [Guide], [Template], [Checklist], [Free], [Updated]
- (With Examples), (Step-by-Step), (Free Download)
- Studies show brackets in titles increase CTR by ~38%

## Character & Pixel Limits

### Title Tag
- **Character limit**: 60 characters (general guideline)
- **Pixel limit**: ~580px on desktop, ~480px on mobile
- **Google behavior**: Truncates with "..." if too long, may rewrite entirely
- **Best practice**: Put most important info in first 50 characters

### Meta Description
- **Character limit**: 155 characters (general guideline)
- **Pixel limit**: ~920px on desktop
- **Google behavior**: May show a different snippet from page content
- **Best practice**: Complete thought in first 120 characters

### Pixel Width Reference (approximate)
| Character Type | Avg Pixel Width |
|---------------|----------------|
| Uppercase (W, M) | 12-15px |
| Lowercase (i, l, t) | 4-6px |
| Average character | 8-9px |
| Space | 3-4px |
| Separator (| — -) | 6-10px |

## A/B Testing Titles

### What to Test
- With vs without year
- With vs without numbers
- Question vs statement format
- With vs without brand name
- Power word variations
- Different separators (| vs — vs :)

### How to Measure
- CTR in Google Search Console (compare 2-4 week periods)
- Aim for minimum 1,000 impressions before drawing conclusions
- Control for ranking position changes
- Test one variable at a time
